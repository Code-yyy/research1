Final Year Research Project

AI-Assisted IoMT Network Attack Detection for Remote Patient Monitoring Systems

This research proposes an edge-based security framework for medical IoT environments by combining secure communication, honeypot-based threat intelligence, lightweight AI-driven attack detection, and automated mitigation for remote patient monitoring systems.

Key Domains

Domain

This section presents the research background, identified gap, project problem, objectives, methodology, and the technologies used in the proposed solution.

Literature Survey

Existing literature shows that IoMT devices are highly vulnerable to cyberattacks due to limited processing power, small memory capacity, and low energy resources. Many current solutions focus only on detection or depend heavily on cloud-based processing.

Research Gap

There is a lack of practical edge-based solutions that combine secure communication, local threat intelligence generation, lightweight detection, automated mitigation, and secure local logging in one integrated framework for medical IoT systems.

Research Problem

Medical IoT devices used in remote patient monitoring are resource-constrained and exposed to attacks such as unauthorized access, data theft, and denial-of-service attacks. Traditional security solutions are often too heavy or too slow for real-time medical environments.

Research Objectives

  • Develop secure lightweight communication for IoMT devices.
  • Create a local edge honeypot to collect attack data.
  • Design a lightweight AI-based attack detection model.
  • Implement automated mitigation and secure local incident logging.

Methodology

The system uses ESP32-based IoMT devices and a Raspberry Pi gateway. Sensitive data is protected using lightweight encryption. A local honeypot captures attack attempts, a lightweight model detects suspicious traffic, and an SDN-inspired mitigation mechanism responds to threats while preserving incident logs securely.

Technologies Used

ESP32 Raspberry Pi Python C / C++ TensorFlow Lite Flask Wireshark Honeypot SDN
